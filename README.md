# DataScience Portfolio

Machine learning projects demonstrating end-to-end data science workflows including EDA, feature engineering, model training, and evaluation.

## Projects

### 1. Credit Scoring Model
**Goal**: Predict credit card default probability to support lending decisions  
**Dataset**: 30,000 credit card clients from UCI  
**Techniques**: Classification, feature engineering, risk assessment  
**Location**: `credit_scoring_model/`

### 2. Flight Delay Prediction
**Goal**: Predict flight delays to improve travel planning  
**Dataset**: Flight delay and cancellation data (2019-2023)  
**Techniques**: Time series analysis, classification  
**Location**: `flight_delay_prediction/`
